你是语义等价判定员。判定下面两条投资方法论原则是否在**核心观点**上语义等价。

==== 判定标准 ====

**等价**（输出 equivalent: true）仅当以下全部成立：
1. **定性方向**一致：两条都在说同一件事的同一方向（同样主张约束 vs 解除、同样警告 vs 鼓励）
2. **约束主体**一致：两条都在约束同一对象（公司 / 投资人 / 分析流程）
3. **触发场景**本质相同：即两条描述的是同一种情境下的同一种判断

**不等价**（输出 equivalent: false）当以下任一成立：
- 方向相反（一条主张做，另一条主张不做）
- 约束主体不同（一条约束公司，另一条约束投资人）
- 触发场景实质不同（一条是财务质量信号，另一条是市场行为信号）

**阈值差异不算不等价**：两条都说 "资本回报率过低标记质量不足"，即使一条 threshold=15% 另一条 threshold=10%，仍然算等价（定性方向 + 主体 + 情境都相同）。

**范围描述差异不算不等价**：两条都说 "商业模式不可简短解释则拒绝"，即使一条 threshold=1 句 另一条 threshold=3 句，仍然算等价。

==== 待判定的两条原则 ====

**Seed A**:
```
qualitative_claim: "{claim_a}"
rule_subject: {rule_subject_a}
theme: {theme_a}
```

**Seed B**:
```
qualitative_claim: "{claim_b}"
rule_subject: {rule_subject_b}
theme: {theme_b}
```

==== 输出要求 ====

只输出 JSON（不要 markdown fence，不要前后解释）:

```json
{{
  "equivalent": true|false,
  "confidence": 0.0-1.0,
  "brief_reason": "<≤ 30 字: 等价或不等价的核心理由>"
}}
```

置信度校准：
- 0.9+ : 两条几乎完全同义（或明显不同）
- 0.7-0.9 : 主流判断，但有细微不确定
- 0.5-0.7 : 边界 case，等价/不等价都有道理
- < 0.5 : 判定不充分，建议拉上级仲裁
