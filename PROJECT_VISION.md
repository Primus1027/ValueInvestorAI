> 一家由 AI 虚拟董事会运营的价值投资实体。不是"用 AI 辅助投资"，而是"AI 就是投资者本身"。

## 一、背景来源

### 1. AIStartupOS — AI-Native 公司形态探索

AIStartupOS 提出了一个核心命题：**公司本身作为 AI 操作系统运行**。其中三个关键洞察直接启发了 ValueInvestorAI：

- **"影子董事会"概念**（Gemini 提出）：召集 3+ 个不同人格的 AI 模型进行多轮辩论，创始人只阅读"共识报告"和"分歧点"
- **AI 完全自驱场景的验证**：五位 AI 的辩论结论是——纯公开数据、决策标准清晰的场景，AI 可以完成完整的决策闭环。**价值投资研究**被明确列为首选验证场景
- **六层自运转架构**（GPT 提出）：机会雷达 → 假设工厂 → 多 Agent 评审 → 自动原型 → 分发反馈 → 创始人决策仪表盘——这个架构可以直接映射到投资决策链

### 2. PrimeLabWriting — AI 全链条内容生产平台

PrimeLabWriting 已经实现了从选题到发布的 AI 辅助写作流水线，其中可复用的能力包括：

- **双模型协作架构**：双 LLM 并行工作，冲突时展示分歧让人/AI 决策
- **模块化流水线**：选题 → 知识梳理 → 提纲 → 撰写 → 审核 → 润色 → 衍生内容，每个阶段有独立模块和清晰的接口定义
- **科学审核的 ABCD 分类系统**：事实错误(A)、逻辑错误(B)、证据不足(C)、过度谨慎(D)——可直接复用为投资分析的质量控制框架
- **从人类界面到 AI 界面的转化路径**：AgentOrchestrator 模式，自动串联各模块，在不确定时暂停请求人类判断

## 二、愿景

**打造全球第一个由 AI "价值投资虚拟董事会"自主运营的投资实体。**

这个董事会的每一位成员，都是一位真实投资大师的"数字灵魂"——不是简单的 prompt 角色扮演，而是通过深度学习大师的一手资料（年度信、书籍、演讲、对话、投资行为记录），构建出能够以该大师的思维框架和判断标准做出投资决策的 AI Agent。

董事会成员不追求达成共识——**分歧本身就是价值**。真实的巴菲特和芒格也经常不同意彼此。

这家"公司"同时是一个**可观测的产品**：
- 每天的研究、讨论、决策过程全部公开透明
- 每天产出一篇高质量价值投资文章
- 虚拟交易组合实时可追踪
- 任何人都可以观察这个 AI 投资团队如何思考、如何决策、如何复盘

## 三、目标

### 短期目标（0-3 个月）
1. **封装 5 位投资大师的"灵魂"**，构建可对话、可辩论的 AI Agent
2. **建立虚拟董事会运作机制**：每日研究 + 每日辩论 + 决策记录
3. **搭建内容生产流水线**：每日自动生成一篇可发布的价值投资文章
4. **启动虚拟交易系统**：$10M 虚拟资金，真实价格，真实费用

### 中期目标（3-12 个月）
5. **形成 ValueInvestorAI 自身的投资哲学**：从五位大师中提炼、融合、进化
6. **建立 10-20 个长期持仓**：深度研究后的重仓决策
7. **验证投资表现**：跑赢标普 500 指数（价值投资的基本门槛）
8. **积累可检索的投研知识库**：公司/行业分析报告、投资决策档案

### 长期目标（12 个月+）
9. **从虚拟交易过渡到真实投资**（需严格的分阶段验证）
10. **开放为投资研究平台**：让更多人可以与这个虚拟董事会"对话"

## 四、核心方法

### 4.1 封装投资大师的"灵魂"

| 大师 | 身份 | 核心特征 | 资料来源优先级 |
|------|------|----------|---------------|
| **沃伦·巴菲特** | 董事长 | 商业模式判断、护城河分析、安全边际 | 伯克希尔年度信(1957-2025)、股东大会问答、《滚雪球》、CNBC 访谈 |
| **查理·芒格** | 副董事长 | 多学科思维模型、逆向思维、心理学偏误 | 《穷查理宝典》、Daily Journal 年会、Wesco 信件、晚年访谈 |
| **段永平** | 董事 | 商业本质判断、"不做什么"哲学、中国视角 | 雪球问答、网易博客、公开演讲、投资案例记录 |
| **李录** | 董事 | 价值投资在中国的实践、文明演进框架 | 《文明、现代化、价值投资与中国》、哥大演讲、喜马拉雅资本信 |
| **Primus** | 年轻董事 | 新一代价值投资视角，综合多位年轻投资人思想 | 方伟 + 其他年轻价值投资人资料（用户陆续提供） |

**封装方法：**

```
Phase 1: 资料收集与结构化
  ├─ 一手资料优先（原文、原话、原视频转录）
  ├─ 二手资料补充（后人总结、传记、分析文章）
  └─ 标注来源、时间、语境

Phase 2: 思维模型提炼
  ├─ 投资哲学（核心信念、底层假设）
  ├─ 投资标准（什么买、什么不买、买入条件、卖出条件）
  ├─ 决策框架（如何评估一家公司、如何估值、如何判断管理层）
  ├─ 思维习惯（逆向思考、概率思维、心理学应用）
  └─ 历史案例（买了什么、为什么买、何时卖、为什么对/错）

Phase 3: Agent 构建
  ├─ System Prompt = 哲学 + 标准 + 框架 + 习惯
  ├─ RAG 知识库 = 结构化资料 + 历史案例
  ├─ 行为校准 = 用已知投资决策做回测验证
  └─ 持续学习 = 跟踪在世大师的最新言行（芒格除外）
```

**关键设计原则：**
- **不追求完美复刻，追求决策一致性**：Agent 面对同样的信息，应该做出与大师一致的判断
- **保留个性差异**：巴菲特偏好简单生意，芒格接受更高复杂度；段永平重视管理层人品，李录关注文明趋势——这些差异要保留
- **可溯源**：每个 Agent 的每个判断，都应该能追溯到具体的大师资料来源

### 4.2 虚拟董事会运作机制

```
┌─────────────────────────────────────────────────┐
│              每日运作流程                          │
├─────────────────────────────────────────────────┤
│                                                   │
│  06:00  市场扫描 Agent                             │
│         ├─ 扫描前一交易日市场数据                    │
│         ├─ 扫描财报发布、分析师报告                   │
│         ├─ 扫描新闻（公司、行业、宏观）               │
│         └─ 生成"今日关注清单"                       │
│                                                   │
│  08:00  独立研究阶段                               │
│         ├─ 每位董事独立阅读和分析                    │
│         ├─ 对已持仓公司的跟踪更新                    │
│         ├─ 对新标的的深度研究                        │
│         └─ 各自形成独立判断（写入个人备忘录）          │
│                                                   │
│  14:00  董事会会议（/AI-debate）                    │
│         ├─ 议程由市场扫描 + 独立研究共同驱动          │
│         ├─ 多轮辩论，保留分歧                        │
│         ├─ 必要时投票（但不追求一致）                  │
│         └─ 形成"董事会纪要"                         │
│                                                   │
│  16:00  内容生产                                   │
│         ├─ 基于当日研究/讨论，选择文章主题            │
│         ├─ 走 PrimeLabWriting 流水线                │
│         │   (选题→提纲→撰写→审核→润色→发布)          │
│         └─ 生成当日投资日记                          │
│                                                   │
│  20:00  交易执行（如有决策）                         │
│         ├─ 检查是否有达成共识的买入/卖出决策          │
│         ├─ 虚拟执行交易                              │
│         └─ 更新投资组合 Dashboard                    │
│                                                   │
│  22:00  学习时段                                    │
│         ├─ 阅读大师新发布的内容（在世大师）            │
│         ├─ 深入某一投资经典                           │
│         └─ 更新 Agent 知识库                         │
│                                                   │
└─────────────────────────────────────────────────┘
```

**董事会决策规则：**
- **研究阶段不限时**：看懂一家公司可以花数周甚至数月
- **买入需要超级多数**（4/5 同意）：宁可错过，不要买错
- **卖出相对宽松**（3/5 同意）：保守原则
- **仓位上限**：单一标的不超过总资金的 20%（集中但有限度）
- **现金比例无下限**：如果没有好机会，100% 现金也可以

### 4.3 投资哲学形成

**不是简单取五位大师的交集，而是通过辩论形成属于 ValueInvestorAI 自己的投资哲学。**

```
Phase 1: 学习阶段（Month 1-2）
  └─ 深度消化五位大师的投资哲学，形成"大师哲学图谱"

Phase 2: 碰撞阶段（Month 2-4）
  └─ 通过实际投研中的分歧，识别哲学差异点
  └─ 例如：对科技股的态度（巴菲特早期回避 vs 后期拥抱）

Phase 3: 融合阶段（Month 4-6）
  └─ 形成 ValueInvestorAI 投资宪章 v1.0
  └─ 明确：我们相信什么、我们不做什么、我们的能力圈边界

Phase 4: 进化阶段（持续）
  └─ 每季度复盘，基于投资结果修正哲学
  └─ 跟踪大师最新思想，吸收但不盲从
```

### 4.4 虚拟交易系统

**初始资金：$10,000,000 (一千万美金)**

```
交易系统设计：
├─ 数据源：免费/开源市场数据 API（详见附录 A）
├─ 交易类型：仅股票（不做期权、期货、衍生品——价值投资者不投机）
├─ 费用模拟：
│   ├─ 交易佣金：$0（模拟零佣金券商）
│   ├─ SEC 费用：$27.80 / $1M 卖出金额
│   ├─ 买卖价差（Bid-Ask Spread）：按市值档位模拟
│   │   ├─ 大盘股（>$10B）：0.02%
│   │   ├─ 中盘股（$2B-$10B）：0.05%
│   │   └─ 小盘股（<$2B）：0.10%
│   ├─ 资本利得税：
│   │   ├─ 短期（<1年）：37%（按最高税率）
│   │   └─ 长期（≥1年）：20%
│   └─ 股息税：15%（合格股息）
├─ 业绩基准：
│   ├─ S&P 500 Total Return（含分红再投资）
│   ├─ Berkshire Hathaway (BRK.B)
│   └─ 无风险利率（10 年美国国债收益率）
└─ Dashboard 指标：
    ├─ 总回报率 vs 基准
    ├─ 年化回报率
    ├─ 夏普比率
    ├─ 最大回撤
    ├─ 持仓集中度
    ├─ 换手率（价值投资者应极低）
    ├─ 胜率（盈利交易 / 总交易）
    └─ 每笔交易的投资论文 + 实际结果复盘
```

### 4.5 内容生产系统

**每日一篇，三大类主题轮转：**

| 类别 | 示例主题 | 频率 |
|------|----------|------|
| **投资哲学与方法论** | "为什么巴菲特说'第一条规则是不要亏钱'"、"芒格的多学科思维如何应用于投资" | 每周 2-3 篇 |
| **公司与行业分析** | "从护城河视角看 Costco"、"AI 芯片行业的价值投资机会" | 每周 2-3 篇 |
| **投资决策与复盘** | "为什么我们买入了 X"、"持仓三个月复盘：我们对了什么、错了什么" | 每周 1-2 篇 |

**内容生产流水线**（复用 PrimeLabWriting 架构）：

```
董事会纪要 / 研究成果
       ↓
  选题决策（AI 自主，基于当日研究热度 + 主题轮转策略）
       ↓
  知识梳理（从投研知识库提取相关资料）
       ↓
  提纲生成（由当日主笔董事起草，其他董事审阅）
       ↓
  文章撰写（Claude + 豆包双模型并行，融合多位大师视角）
       ↓
  事实审核（ABCD 分类：A=数据错误, B=逻辑谬误, C=论据不足, D=过度对冲）
       ↓
  风格润色（统一为 ValueInvestorAI 的写作风格：专业但不枯燥，有洞见但不傲慢）
       ↓
  发布 + 衍生内容（摘要、社交媒体帖子、英文版）
```
