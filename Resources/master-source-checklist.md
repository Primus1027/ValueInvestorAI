## 资料收集标准框架

### 信息源类型与优先级

| 优先级 | 类型 | 定义 | AI 学习价值 |
|--------|------|------|-----------|
| **P0** | 决策时点原文 | 大师本人在做出投资决策前后亲自写作，时间接近决策 | 极高 — 唯一能捕捉真实思维结构的资料 |
| **P1** | 演讲/访谈原始转录 | 逐字转录（非摘要），捕捉类比思维和即兴回答 | 高 — 追问中暴露真实判断逻辑 |
| **P2** | 投资行为记录 | 可追溯到具体时间点的买卖行为数据 | 高（行为验证）/ 低（不携带理由） |
| **P3** | 决策同期背景 | 大师做决策时市场上能看到的信息（年报、研报、宏观数据） | 中高 — 还原信息集合，理解筛选过程 |
| **P4** | 第三方传记 | 经独立核实的深度研究，提供大师不会公开说的信息 | 中 — 需标注"叙事重构" |
| **排除** | 二手解读/教学 | 原则总结、投资教程、无出处语录 | 不可用 — 平滑化删除了矛盾处和例外处 |

### 核心排序原则

1. **决策时点接近度优先**：1989 年谈可口可乐 > 2015 年回忆可口可乐
2. **一手性优先但分场合**：内部文件 > 致股东信（有法律责任）> 演讲（有表演成分）> 媒体采访（戏剧性选择）
3. **投资决策相关 >> 人生哲学**：谈具体公司/估值/买卖时机 > 谈商业本质/护城河 > 谈友谊/健康/道德
4. **大师说"不"的案例 > 成功案例**：拒绝案例展示真实过滤标准，无幸存者偏差
5. **失败案例 > 成功案例**：错误展示判断边界，是学习价值最高的素材
6. **跨时间段对比**：同一原则在不同时期的表述变化 = 风格演化证据

### 思想演变的两种类型（决定权重的关键框架）

大师的思考在持续进化。资料中同一大师在不同时期的言行可能矛盾，但矛盾的性质决定了处理方式：

**类型 A：背景变了，决策都对（context-dependent，并列权重）**

两个阶段的决策都是正确的，只是适用条件不同。AI 必须保留两者，但加上阶段标签。

```
示例：巴菲特的资金规模 vs 策略变化
  1960s ($10M)：烟蒂股、套利、控制权投资 → 小规模时正确
  2010s ($500B+)：只买大市值、高流动性 → 大规模时正确
  → 不矛盾，是同一投资哲学在不同约束条件下的表现
  → AI 标注：stage=early_partnership / stage=late_berkshire
```

处理原则：
- 每条资料必须标注 **阶段标签**（时间 + 管理规模 + 当时的核心策略）
- AI 分析时需明确"我在模拟哪个阶段的大师"
- 不同阶段的资料并列存储，权重相等

**类型 B：大师自己纠错，以最新反思为准（correction chain，新覆盖旧）**

大师后来认为之前的做法是错误的，改变了原则。这时要以修正后的版本为主要权重。

```
示例：巴菲特对"永远持有"的修正
  1988 原始立场："可口可乐是永远持有的股票"
  ~2000s 事件：可口可乐估值飙升到 50x PE
  巴菲特后来反思："1998年估值疯狂的时候是应该卖的，当时太乐观了"
  → 修正后原则：永远持有的前提是估值仍合理，不是无条件持有
  → AI 标注：correction_chain, 最终权重给修正版
```

处理原则：
- 建立 **correction chain**：原始立场 → 发生了什么 → 大师的反思 → 修正后的原则
- 最终权重给修正后的版本
- 但保留完整链条（"为什么错了"本身是极高价值的学习素材）
- `post-hoc` 标签中需区分：`post-hoc-rationalization`（美化成功）vs `post-hoc-correction`（承认错误，价值极高）

**辨别方法：** 如果大师说"当时就是对的，现在换个做法也是对的" → 类型 A。如果大师说"当时做错了" → 类型 B。

### 质量标准

**合格资料必须满足：**
- 可溯源（具体信件日期、演讲场合、访谈媒体）
- 时间戳清晰（能与投资决策时间线对应）
- 完整上下文（不是孤立语录，含前后文）

**需标注降权的资料：**
- `post-hoc` — 事后归因（已知结果后的解释）
- `unverified` — 未注明来源的语录
- `translation-risk` — 翻译版本（关键术语可能失真）
- `edited-excerpt` — 媒体剪辑版（完整语境被删除）

### AI 学习的特殊陷阱

| 陷阱 | 说明 | 规避方法 |
|------|------|---------|
| 过度精炼的原则总结 | "巴菲特十条原则"删除了模糊性和例外 | 用原文训练，不用总结版 |
| 高光演讲 | TED/毕业典礼是策划过的，不是真实判断 | 优先用即兴问答和私下通信 |
| 单一成功案例过度拟合 | 可口可乐案例 ≠ 所有消费品都可投 | 确保案例多样性 |
| 不同阶段混合训练 | 巴菲特 1960s 烟蒂股 vs 2000s 护城河看似矛盾但都对 | 类型 A 处理：打阶段标签，按背景区分，不混合 |
| 把大师已纠错的旧原则当真理 | "永远持有"已被巴菲特自己修正为"估值合理时持有" | 类型 B 处理：建 correction chain，新覆盖旧 |
| 把事后反思和事后美化混为一谈 | 大师承认错误（极高价值）vs 美化成功（需降权）是完全不同的 | 区分 post-hoc-correction vs post-hoc-rationalization |
| 中文互联网伪段永平 | 大量转载/改写/伪造 | 以雪球原帖时间戳为唯一可信源 |
